Visão Geral

Resumo do Projeto

O projeto Express Marketplace, desenvolvido na disciplina de Engenharia de Software, teve como objetivo a criação de uma plataforma de e-commerce especializada em tecnologia, voltada para a venda de produtos eletrônicos. A proposta do projeto foi desenvolver um marketplace digital, permitindo que usuários pudessem comprar e vender produtos. O intuito principal é educacional: aplicar e aprimorar os conhecimentos adquiridos ao longo do curso. Vale ressaltar que o site não será disponibilizado para transações reais, servindo exclusivamente como uma experiência de aprendizado.

Conheça o Projeto

Tecnologias Utilizadas

O projeto utilizou PHP, Node.js e banco de dados MySQL, explorando tanto back-end quanto front-end, com o uso de HTML, CSS e JavaScript para interfaces interativas.

Prototipagem

O design foi prototipado no Figma, com foco em garantir uma experiência de usuário intuitiva e acessível, alinhada às necessidades do projeto.

Funcionalidades

O sistema implementa CRUD, controle de login e sessão, APIs, testes unitários e utiliza a arquitetura MVC, com papéis de Vendedor e Usuário, além de um dashboard exclusivo para vendedores.

Metodologias Ágeis

Foram aplicadas práticas como o GitFlow e conceitos de métodos ágeis, como o Agile Cycle, para organizar o desenvolvimento.

Conclusão

O projeto culminou na entrega de uma aplicação refinada, que atende aos requisitos definidos e está documentada com todos os artefatos finais necessários. Embora não tenha sido implementada a segurança necessária para uso comercial, a base do sistema é funcional e está pronta para aprimoramentos futuros.
